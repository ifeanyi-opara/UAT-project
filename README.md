# Single-User OFDMA Transceiver with Adaptive QC-LDPC and Control Signaling

A robust, modular, end-to-end Python implementation of a single-user Orthogonal Frequency Division Multiple Access (OFDMA) transceiver framework matching key 802.11ax (Wi-Fi 6) paradigms. This system features dynamic Link Adaptation, systematic Quasi-Cyclic Low-Density Parity-Check (QC-LDPC) forward error correction, cyclic-prefix (CP) guided Carrier Frequency Offset (CFO) estimation, and an Iterative Clipping and Filtering (ICF) loop for Peak-to-Average Power Ratio (PAPR) reduction.

It natively supports both high-fidelity software simulations (AWGN loopback) and direct deployment on physical Software Defined Radio (SDR) hardware via the **Ettus UHD** Python bindings interface (e.g., USRP B200mini).

---

## 🛠 Features Matrix

* **Dynamic PHY Configuration:** Automatic link adaptation spanning multiple modulation modes (`QPSK`, `16QAM`, `64QAM`, `256QAM`) and structured toy code rates matching explicit hardware limits ($R = 1/12, 1/6, 1/3, 1/2, 2/3, 5/6$).
* **Systematic QC-LDPC Matrix Generation:** Generates Lifted Parity Check ($H$) matrices using a circulant permutation algorithm over configurable block size metrics ($Z$), complemented by Bit-Flip / Linear Approximation Soft Decoding via `ldpc`.
* **Joint Sync & Signaling:** Transmits an integrated preamble embedding a root Zadoff-Chu sequence for synchronization and cyclically-shifted control structures to blind-decode current channel coding states.
* **PAPR Mitigation (ICF):** Features an advanced Iterative Clipping and Filtering runtime pipeline allowing targeted compression on full-scale digital waveforms prior to DAC upconversion.
* **UHD Multi-Threading Architecture:** Complete execution thread isolation for transmitting and receiving streaming frames via raw hardware calls.

## 📐 DSP Pipeline & Mathematical Architecture

### 1. Systematic Forward Error Correction & Integrity

Data structures undergo nested validation. Payload frames are prepended and wrapped inside a standard 16-bit Cyclic Redundancy Check (CRC-CCITT) generated by the generator polynomial:


$$G(x) = x^{16} + x^{12} + x^5 + 1$$

The QC-LDPC generator matrix construction builds code segments systematic over the matrix equations:


$$G = [I_K \mid A^T]$$


where $A$ constitutes the left portion of the derived target lifted parity check block array $H$.

### 2. Frequency Tracking & Fine Compensation

Carrier Frequency Offset (CFO) recovery estimates temporal shifts by exploiting the cyclic phase repetitions between the localized symbol prefix and boundary trailing components:


$$\hat{f}_{\text{cfo}} = -\frac{\angle \left( \sum_{n=0}^{N_{\text{cp}}-1} y[n] \cdot y^*[n + N_{\text{fft}}] \right)}{2\pi} \cdot \frac{f_s}{N_{\text{fft}}}$$

### 3. Channel Estimation & MMSE Equalization

Frequency domain equalization implements an Minimum Mean Square Error (MMSE) estimator layout mapping linear channel coefficients $H_k$ across subcarriers via linear interpolation profiles from explicit pilot nodes:


$$\hat{X}_k = \frac{Y_k \cdot \hat{H}_k^*}{|\hat{H}_k|^2 + \frac{1}{\text{SNR}_{\text{linear}}}}$$

## ⚡ Quick Start & Prerequisites

### 1. Installation Requirements

Ensure your host operating system environment includes functional libraries for `UHD` (USRP Hardware Driver) matching your hardware peripherals. Install your standard Python package array using the following configuration:

```bash
pip install numpy matplotlib numba ldpc-py-tools

```

> *Note: Ensure your `ldpc` workspace library includes path structures pointing to the localized `BpLsdDecoder` setup.*

### 2. Running Simulations

By default, the script main workspace initiates in a purely simulated framework loopback mode (`SIMULATE = True`), requiring zero attached physical SDR targets:

```bash
python3 UAT.py

```

### 3. Physical USRP Deployment

To route processing pipelines straight over active RF mixers via UHD frontend boards, modify the underlying logic gates located inside `main()`:

```python
SIMULATE = False  # Activates live MultiUSRP streaming channels

```
